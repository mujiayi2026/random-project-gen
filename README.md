# 🎲 Random Project Generator

帮你找到下一个 vibecoding 项目！

## 使用方法

```bash
# 生成 1 个创意
python3 gen.py

# 生成 5 个创意
python3 gen.py -n 5

# 只看高难度的
python3 gen.py --hard

# 保存到 ideas.md
python3 gen.py --save
```

## 特性

- 🎯 随机组合：技术栈 × 项目类型 × 领域 × 难度
- ✨ 附带 Twist：给项目加点独特的挑战
- 📦 建议项目名：省去起名的烦恼
- 💾 保存功能：记录你的灵感

## 数据库

- 15 种技术栈
- 12 种项目类型
- 15 种领域主题
- 4 种难度等级
- 15 种 Twist

总共可生成 **162,000** 种不同组合！
